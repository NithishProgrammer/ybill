**Ybill** is a high-speed, privacy-first web application designed for instant invoice generation. Built for small businesses, freelancers, and shops who need professional bills without the overhead of accounts, subscriptions, or complex databases.

---

### 🔗 Live Access
**Use it now:** [https://nithishprogrammer.github.io/ybill](https://nithishprogrammer.github.io/ybill)

---

### 🌟 Key Features
* **🚀 Zero Signup:** No registration required. Open the page and start billing instantly.
* **🔒 Privacy First:** Your data never leaves your computer. All business profiles and invoice history are saved in your browser's `localStorage`.
* **📄 High-Quality PDF:** Generates crisp, well-structured, and print-ready PDF invoices.
* **🔄 Full CRUD Support:** Create, Read (History), Edit, and Delete past bills easily.
* **📱 Responsive UI:** Built with Tailwind CSS for a seamless experience on mobile and desktop.

### 🛠️ Tech Stack
- **Frontend:** HTML5, Tailwind CSS
- **PDF Engine:** [jsPDF](https://github.com/parallax/jsPDF) & [html2canvas](https://github.com/niklasvh/html2canvas)
- **Storage:** Browser LocalStorage API (Client-side persistence)

### 🤖 AI-Powered Development
**Ybill is an AI-augmented project.** It was conceptualized and co-developed using **Google Gemini**, leveraging AI to:
- Architect the high-fidelity PDF rendering logic.
- Optimize state management for CRUD operations without a backend.
- Design the minimalist "Ybill" brand identity and responsive layout.
